Instrucciones

adelante / ade

Ejemplo: adelante 50 movera el robot 50 pasos hacia el frente.

atras / atr

Ejemplo: atras 50 movera el robot 50 pasos hacia atrás.

derecha / der

Ejemplo: derecha 45 girará el robot 45 grados a la derecha.

izquierda / izq

Ejemplo: izquierda 45 girará el robot 45 grados a la derecha.

color

Ejemplo: color azul cambia el color de las lineas a a.zul

levantar_pluma / lp

Ejemplo: levantar_pluma El robot traza una linea cuando se mueve.

bajar_pluma / bp

Ejemplo: bajar_pluma El robot puede moverse sin trazar una línea.

centrar / ctr

Ejemplo: centrar coloca el robot en el centro de la pantalla.

limpiar

Ejemplo: limpiar remueve todos los trazos por el robot.

repetir

Ejemplo: repetir 100 [adelante 200 derecha 89] hará un círculo genial
